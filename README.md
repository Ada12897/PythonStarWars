# Star Wars Game

Учебный проект на Python с использованием Pygame. Простая аркадная игра в стиле Star Wars.  

## Описание
Это учебная игра, сделанная для практики работы с Pygame. Проект не завершён и служит для отработки базовых навыков программирования игр.  

## Технологии
- Python 3.x  
- Pygame
